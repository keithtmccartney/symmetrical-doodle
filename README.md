* Install Sass @ [https://sass-lang.com/install](https://sass-lang.com/install); Done;
* Check Ruby Sass version @ "sass -v" on a Ruby-based terminal; Done;

# Instructions
* Create a "css" and a "scss" directory; Done;
* On a Ruby-based terminal, run a watch command to have Sass monitor the directories above "sass --watch scss:css"; Done;
* Make changes to the "css" stylesheet, the terminal will monitor any changes (see below for example); Done;
>>> Sass is watching for changes. Press Ctrl-C to stop.
      write css/style.css
      write css/style.css.map
>>> Change detected to: scss/style.scss
      write css/style.css
      write css/style.css.map
